# Prompt Engineering for Water Monitoring

This module is a prototype prompt-engineering extension for the existing IoT water monitoring dashboard.

## Goal
Convert sensor readings such as temperature, pH, and turbidity into:
- risk alerts
- natural-language explanations
- practical recommendations
- user-friendly Q&A responses

## Input
- Temperature
- pH
- Turbidity

## Output
- Risk level
- Explanation
- Recommended action

## Status
Prototype using simulated prompt-response workflow.
